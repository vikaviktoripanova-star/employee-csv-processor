# Лабораторная работа: Обработка CSV файлов с данными о сотрудниках

## Описание проекта

Проект представляет собой Java приложение для чтения и обработки данных о сотрудниках из CSV файлов. Приложение преобразует данные из CSV формата в объекты Java и предоставляет статистическую информацию.

## Цели работы

- Реализовать чтение данных из CSV файла
- Создать объектную модель для представления данных
- Обеспечить обработку ошибок и валидацию данных
- Написать unit-тесты для проверки функциональности
- Создать профессиональную документацию к коду

## Архитектура проекта

## Формат данных CSV
```
CSV файл должен содержать следующие поля (разделитель - ;):

| Поле | Тип | Формат | Описание |
|------|-----|---------|----------|
| ID | Integer | - | Уникальный идентификатор |
| Name | String | - | Имя сотрудника |
| Gender | String | "Male"/"Female" или "M"/"F" | Пол |
| BirthDate | Date | "dd.MM.yyyy" | Дата рождения |
| Division | String | - | Подразделение |
| Salary | Double | - | Зарплата |
```

**Пример строки:**
```
28281;Aahan;Male;15.05.1970;I;4800
```

## Запуск приложения

### Требования:
- Java 11 или выше
- Maven 3.6+

### Сборка и запуск:

# Клонирование репозитория
git clone <repository-url>
cd employee-csv-processor

# Сборка проекта
mvn clean compile

# Запуск приложения
mvn exec:java -Dexec.mainClass="com.example.Main"

# Запуск тестов
mvn test

## Функциональность

### Основные возможности:

1. **Чтение CSV файлов** - поддержка различных форматов данных
2. **Автоматическая дедупликация** - подразделения с одинаковыми именами представляются одним объектом
3. **Валидация данных** - проверка корректности форматов
4. **Статистика** - подсчет сотрудников по полу, уникальных подразделений, зарплат
5. **Обработка ошибок** - детализированные сообщения об ошибках
